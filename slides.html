Let's review how a basic Android project looks like

Hands-on

Create a new Android project - Archer

  • Src folder - The source folder
  • Gen folder - Generated stuff. Don't touch. Managed by ADT.
  • Bin folder - The compiled stuff. Useless to touch. Managed by ADT.
  • Lib folder - JAR files are put here
  • Res folder - Put the application resources here
  • drawables folder - Image files
  • layout folder - Layout files
  • menu folder - XML configuration files for menus
  • values folder - XML configuration files for strings, styles (the CSS version of Android), dimensions, etc

The suffix hdpi, ldpi, mdpi, etc. correspond to screen dimensions. Remember that unlike iOS, Android is installed on phones with different screen sizes.

AndroidManifest.xml - important configuration file for your Android application

The basic android building blocks are

  • Activity
  • Broadcast Receiver
  • Service
